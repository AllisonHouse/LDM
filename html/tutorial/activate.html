How to activate your LDM system.

Once the LDM system is installed and configured, activating it takes only a few steps. See LDM Reference: Activating the LDM Installation for instructions.

You should view the LDM log file, $LDMHOME/logs/ldmd.log, to see what a successful activation looks like. Information on interpreting the LDM log file can be found at LDM Reference: LDM Log File Format and Interpretation.
